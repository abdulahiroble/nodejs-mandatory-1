Tools

Brugt i NodeJS

NPM


Node Package Manager (NPM), er en pakke håndterings manager i JavaScript. Det kan sammenlignes til med "Maven" i Java, som henter de nødvendige dependencies for at kunne køre applikationen


VSCODE


VSCODE er en IDE/Editor, lavet af Microsoft, for Windows, Linux og MacOS. Det kan bruges til mange programmeringssprog, såsom JaveScript, Python, C++, NodeJS mm.


Terminal


Terminalen bruger vi til at installere NodeJS på Windows og hente diverse packages ned, for at kunne køre vores applikation. Derudover bruger vi den også til at starte vores server, samt til "pushe" og "pull" fra GitHub. I terminalen har vi forskellige kommandoer, som vi kører som kan ses på (LINK).


Postman


Postman er et værktøj, der giver dig de basale kompetencer der skal til for at teste webservices / API’er og lave en effektiv integrationstest.
